패킷 파서 상태 머신 (State Machine)

수신 바이트를 한 글자씩 처리하여 완전한 패킷을 조립한다. 상태를 유지하므로 바이트가 여러 번에 걸쳐 도착해도 정확히 파싱된다.

패킷 구조 복습

STX0xAA LENCMD+PLD 바이트 수 CMD명령 코드 PAYLOAD0 ~ N 바이트 CHKCMD⊕PLD XOR ETX0x55

상태 전이 다이어그램

PS_WAIT_STX
← 초기 상태 / 에러 후 리셋
↓ 0xAA 수신
PS_WAIT_LEN
→ LEN = 0 또는 > 9 : NACK(BAD_LEN) 후 PS_WAIT_STX 로 리셋
↓ 유효한 LEN 수신 → payloadLen = LEN−1
PS_WAIT_CMD
↓ CMD 수신
PS_WAIT_PAYLOAD
← payloadLen > 0 인 경우 진입
   payload[idx++] 저장 · idx == payloadLen 되면 다음 상태
   (payloadLen == 0 이면 이 상태를 건너뛰고 바로 PS_WAIT_CHK)
↓ N 바이트 수신 완료
PS_WAIT_CHK
↓ CHK 바이트 저장
PS_WAIT_ETX
→ 0x55 아닌 값 수신 : NACK(BAD_CHK) 후 PS_WAIT_STX 로 리셋
↓ 0x55 수신 → 체크섬 검증 → dispatchPacket()
패킷 처리 완료 → PS_WAIT_STX
※ 재동기화 규칙 : PS_WAIT_STX 상태에서 0xAA 이외의 모든 바이트는 무시된다. 노이즈로 중간 바이트가 손상되어도 다음 패킷의 0xAA가 올 때까지 기다린 뒤 자동으로 재동기화된다.
※ 체크섬 검증 : ETX 확인 후 CHK == CMD ⊕ PAYLOAD[0] ⊕ … ⊕ PAYLOAD[N-1] 을 검사한다. 불일치 시 NACK(0x01) 을 반환한다.

상태 전이 요약표

현재 상태 수신 조건 다음 상태 동작
PS_WAIT_STX byte == 0xAA PS_WAIT_LEN —
PS_WAIT_STX byte ≠ 0xAA PS_WAIT_STX 무시 (노이즈 폐기)
PS_WAIT_LEN 1 ≤ byte ≤ 9 PS_WAIT_CMD len 저장, payloadLen = byte−1
PS_WAIT_LEN byte == 0 또는 > 9 PS_WAIT_STX NACK(BAD_LEN) 전송
PS_WAIT_CMD any byte PS_WAIT_PAYLOAD
또는 PS_WAIT_CHK		 cmd 저장 / payloadLen==0 이면 CHK로 점프
PS_WAIT_PAYLOAD any byte (×N) PS_WAIT_CHK (N번째) payload[] 버퍼에 순차 저장
PS_WAIT_CHK any byte PS_WAIT_ETX chkRecv 저장
PS_WAIT_ETX byte == 0x55 PS_WAIT_STX dispatchPacket() 호출 (체크섬 검증 포함)
PS_WAIT_ETX byte ≠ 0x55 PS_WAIT_STX NACK(BAD_CHK) 전송

코드 구현 — AppProtocol_Feed()

1ms 태스크에서 수신 바이트마다 한 번씩 호출된다. ch = 0(HC-12) / 1(HM-10 BLE).

void AppProtocol_Feed(uint8 byte, uint8 ch)
{
    Parser *p = &s_parser[ch];

    switch (p->state)
    {
    case PS_WAIT_STX:
        if (byte == PROTO_STX)          /* 0xAA */
            p->state = PS_WAIT_LEN;
        /* else: 노이즈 바이트 무시 */
        break;

    case PS_WAIT_LEN:
        if (byte == 0u || byte > (PROTO_MAX_PAYLOAD + 1u))
        {
            p->state = PS_WAIT_STX;
            sendNack(NACK_ERR_LEN, ch);    /* 범위 초과 → NACK */
        }
        else
        {
            p->len        = byte;
            p->payloadLen = byte - 1u;    /* LEN - CMD(1) = 실제 페이로드 수 */
            p->payloadIdx = 0u;
            p->state      = PS_WAIT_CMD;
        }
        break;

    case PS_WAIT_CMD:
        p->cmd   = byte;
        p->state = (p->payloadLen == 0u) ? PS_WAIT_CHK : PS_WAIT_PAYLOAD;
        /* payloadLen == 0 (PING 등) 이면 PAYLOAD 상태 건너뜀 */
        break;

    case PS_WAIT_PAYLOAD:
        p->payload[p->payloadIdx++] = byte;
        if (p->payloadIdx >= p->payloadLen)
            p->state = PS_WAIT_CHK;
        break;

    case PS_WAIT_CHK:
        p->chkRecv = byte;
        p->state   = PS_WAIT_ETX;
        break;

    case PS_WAIT_ETX:
        if (byte == PROTO_ETX)          /* 0x55 */
            dispatchPacket(p, ch);      /* 체크섬 검증 + 명령 실행 */
        else
            sendNack(NACK_ERR_CHK, ch); /* ETX 불일치 */
        p->state = PS_WAIT_STX;         /* 항상 리셋 */
        break;

    default:
        p->state = PS_WAIT_STX;
        break;
    }
}

체크섬 검증 — dispatchPacket() 내부

ETX까지 수신 완료된 후, CMD와 PAYLOAD 전체를 XOR하여 수신된 CHK와 비교한다.

static void dispatchPacket(const Parser *p, uint8 ch)
{
    uint8 chkCalc = p->cmd;
    uint8 i;

    for (i = 0u; i < p->payloadLen; i++)
        chkCalc ^= p->payload[i];      /* CHK = CMD ⊕ PLD[0] ⊕ … ⊕ PLD[N-1] */

    if (chkCalc != p->chkRecv)
    {
        sendNack(NACK_ERR_CHK, ch);    /* 체크섬 불일치 */
        return;
    }

    switch (p->cmd)
    {
    case CMD_MOVE:  /* 0x01 : dir(1B) + speed(1B) */
        g_vehicleCmd   = p->payload[0];
        g_vehicleSpeed = (float32)p->payload[1];
        g_lastMoveTime = g_1ms_counter;
        sendAck(CMD_MOVE, ch);
        break;
    case CMD_MODE:  /* 0x02 : mode(1B) */
        g_vehicleMode = p->payload[0];
        sendAck(CMD_MODE, ch);
        break;
    case CMD_PING:  /* 0x10 : no payload */
        sendAck(CMD_PING, ch);
        break;
    default:
        sendNack(NACK_ERR_CMD, ch);
        break;
    }
}
