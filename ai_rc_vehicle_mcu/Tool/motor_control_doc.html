모터 제어 동작 설계

4WD 차량의 좌/우측 모터 방향을 독립 제어하여 전진·후진·제자리 회전을 구현한다.

동작 FL · RL (좌측) FR · RR (우측) 결과
전진 + 속도 + 속도 직진
후진 − 속도 − 속도 후진
제자리 좌회전 − 속도 + 속도 우측 바퀴 전진, 좌측 바퀴 후진 → 좌회전
제자리 우회전 + 속도 − 속도 좌측 바퀴 전진, 우측 바퀴 후진 → 우회전
정지 Coast Coast 관성 정지 (브레이크 없음)

※ 전륜(FL·FR)에는 후륜 대비 86 % 보정 계수를 적용한다 (spdF = speed × 0.86). 전·후륜 구동력 차이를 보상하기 위함이다.

코드 구현 — AppVehicle_Update()

스케줄러가 10 ms 주기로 호출하며, 전역 변수 g_vehicleCmd·g_vehicleSpeed를 읽어 모터 듀티를 설정한다.

void AppVehicle_Update(void)
{
    sint16 spd  = (sint16)g_vehicleSpeed;
    sint16 spdF = (sint16)(g_vehicleSpeed * 0.86f);  /* 전륜 보정 86 % */

    switch ((VehicleCommand)g_vehicleCmd)
    {
    case VEHICLE_FORWARD:
        DrvMotor_SetDuty(MOTOR_FL,  spdF);   /* 좌전륜  +속도 */
        DrvMotor_SetDuty(MOTOR_FR,  spdF);   /* 우전륜  +속도 */
        DrvMotor_SetDuty(MOTOR_RL,  spd);    /* 좌후륜  +속도 */
        DrvMotor_SetDuty(MOTOR_RR,  spd);    /* 우후륜  +속도 */
        break;

    case VEHICLE_REVERSE:
        DrvMotor_SetDuty(MOTOR_FL, -spdF);   /* 좌전륜  −속도 */
        DrvMotor_SetDuty(MOTOR_FR, -spdF);   /* 우전륜  −속도 */
        DrvMotor_SetDuty(MOTOR_RL, -spd);    /* 좌후륜  −속도 */
        DrvMotor_SetDuty(MOTOR_RR, -spd);    /* 우후륜  −속도 */
        break;

    case VEHICLE_SPIN_LEFT:
        DrvMotor_SetDuty(MOTOR_FL, -spdF);   /* 좌전륜  −속도 */
        DrvMotor_SetDuty(MOTOR_FR,  spdF);   /* 우전륜  +속도 */
        DrvMotor_SetDuty(MOTOR_RL, -spd);    /* 좌후륜  −속도 */
        DrvMotor_SetDuty(MOTOR_RR,  spd);    /* 우후륜  +속도 */
        break;

    case VEHICLE_SPIN_RIGHT:
        DrvMotor_SetDuty(MOTOR_FL,  spdF);   /* 좌전륜  +속도 */
        DrvMotor_SetDuty(MOTOR_FR, -spdF);   /* 우전륜  −속도 */
        DrvMotor_SetDuty(MOTOR_RL,  spd);    /* 좌후륜  +속도 */
        DrvMotor_SetDuty(MOTOR_RR, -spd);    /* 우후륜  −속도 */
        break;

    default: /* VEHICLE_STOP */
        DrvMotor_Coast(MOTOR_FL);
        DrvMotor_Coast(MOTOR_FR);
        DrvMotor_Coast(MOTOR_RL);
        DrvMotor_Coast(MOTOR_RR);
        break;
    }
}
