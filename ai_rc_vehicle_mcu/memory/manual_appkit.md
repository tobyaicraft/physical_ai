# Application Kit TC2X7 하드웨어 매뉴얼 요약

> 원본: Application Kit TC2X7 Hardware Manual V1.0 (2015-04), Infineon Technologies AG

---

## 1. 보드 개요

Application Kit TC2X7은 Infineon TriCore 기반 평가 보드로, TC237/TC267/TC277/TC297 마이크로컨트롤러를 지원한다.

### 1.1 주요 사양

| 항목 | 사양 |
|------|------|
| MCU | TC2X7 (TC237, TC267, TC277, TC297), LFBGA-292 패키지 |
| 보드 크기 | 100mm x 100mm |
| 디스플레이 | LCD XGA 320x240 (ILI9341 컨트롤러) |
| SD 카드 | Mini SD 카드 슬롯 (SPI 모드) |
| CAN | High Speed CAN 트랜시버 (TLE 7250 GVIO) |
| LIN | LIN 트랜시버 (TLE 7259-3GE) |
| 이더넷 | Gigabit PHY PEF7071 (이더넷 지원 CPU에만 장착) |
| USB-UART | FT2232HL (FTDI) 브릿지 |
| 크리스탈 | 20MHz (기본값, Y101 교체 가능) |
| 디버거 | USB miniWiggler JDS 온보드 |
| LED | 4개 범용 LED + 4개 상태 LED |
| RTC | MCP79511 (TC237) / MCP79411 (기타 CPU) |
| 비퍼 | 전자음향 변환기 (P33.0, 2048Hz) |

### 1.2 주요 IC 구성

| IC | 기능 | 비고 |
|----|------|------|
| TLF35584QV | Multi Voltage Safety Micro Processor Supply | 전원 관리 |
| TLE 7250 GVIO | High Speed CAN 트랜시버 | CAN0 |
| TLE 7259-3GE | LIN 트랜시버 | ASCLIN1 |
| FT2232HL | USB to UART 브릿지 | ASC0 + miniWiggler |
| PEF7071 | Gigabit Ethernet PHY (Lantiq) | TC237L 미지원 시 미장착 |
| ADS7843 | 터치스크린 컨트롤러 | QSPI0, SLSO9 |
| XC9572XL | Xilinx CPLD | |
| MCP79511/MCP79411 | RTC + EUI-48 MAC ID | QSPI3 (TC237) / I2C (기타) |
| ILI9341 | LCD 디스플레이 컨트롤러 | SPI-Parallel 변환기 경유 |

## 2. 전원 공급

### 2.1 전원 입력 방식

마이크로컨트롤러에 필요한 3개 전압: +3.3V, +5V (ADC/CAN용), +1.3V (코어, MCU 자체 생성)

#### 방법 1: 전원 커넥터 (X101)
- 입력 범위: **+5.5V ~ +40V DC**
- 권장: 6V / 500mA 이상
- 표준 전원 어댑터 사용 가능 (중앙 양극, 외측 GND)

#### 방법 2: Micro USB (BU301)
- USB를 통한 5V 전원 공급
- TC237 기준 약 400mA 소비 (TC297 등은 더 높음)
- USB 2.0 포트 (500mA) 또는 USB 3.0 포트 (900mA) 권장
- **대부분의 USB 허브(100mA)는 사용 불가**
- 휴대폰 충전기(5V Micro USB) 사용 가능

#### 전원 우선순위
- 전원 커넥터 전압이 USB 5V보다 높으면 전원 커넥터가 우선

### 2.2 TLF35584QV 전원 관리 IC

| 기능 | 사양 |
|------|------|
| 입력 전압 | 3.0V ~ 40V |
| LDO_Com | 5.0V / 200mA (통신용) |
| LDO_C | 3.3V/600mA (VS2) 또는 5.0V/600mA (VS1) |
| Volt_Ref | 5.0V +/-1%, 150mA (ADC용) |
| LDO_Stby | 3.3V/10mA (VS2) 또는 5.0V/10mA (VS1) (대기용) |
| Tracker 1, 2 | 센서 공급, 각 150mA (미사용) |
| 워치독 | 기능적/윈도우 워치독 (설정 가능) |
| SPI 연결 | QSPI2, 칩셀렉트 QSPI2_SLSO1 (P14.2) |

**TC237**: 3.3V 버전 (VS2) 사용  
**TC267/TC277/TC297**: 5.0V 버전 (VS1) 사용

TLF35584는 ISO26262 호환, AEC 인증 완료.

### 2.3 전원 신호 요약

| 신호명 | 설명 |
|--------|------|
| VCC_IN | 공급 입력 (5.5V ~ 40V) |
| VIN | 전원 IC 입력 전압 |
| GND | 접지 |
| VDD | 코어 공급 전압 (1.3V) |
| V_UC | 포트 공급 전압 (TC237: 3.3V / 기타: 5V) |
| VDDM | ADC 아날로그부 공급 (VREF_5V 또는 3.3V) |
| VAREF1/2 | ADC 기준 전압 1/2 |
| VDD_FT | FT2232HL 공급 (3.3V) |
| VDD_USB | USB 공급 전압 (5V) |

## 4. 디버그/프로그래밍 인터페이스

### 4.1 miniWiggler JDS (온보드)

- USB를 통한 저비용 JTAG 디버그 도구
- DAS Server 'UDAS'를 통해 디버깅 수행
- **반드시 최신 DAS 소프트웨어를 PC에 먼저 설치** 후 보드 연결
- 연결 성공 시 **ACTIV LED (D302, 녹색)** 점등
- RUN LED (D301, 파란색): DAS Server에 의해 제어됨

### 4.2 OCDS1 (외부 디버거)

- X301 (IDC16) 커넥터에 외부 디버거 연결 가능
- **miniWiggler JDS와 동시 사용 불가**
- 외부 디버거 연결 전 반드시:
  - ACTIV LED가 꺼져 있는지 확인
  - DAS Server 'UDAS' 중지 및/또는 USB 연결 해제

### 4.3 DAP (Device Access Port)

- FTSH10 10핀 커넥터
- DAP0 (TCK), DAP1 (TMS) 핀 사용

### 4.4 시리얼 연결 (ASC0)

- USB 연결 시 FT2232HL을 통해 자동으로 COM 포트 생성
- ASCLIN0 (P14.0: TX, P14.1: RX)를 통한 시리얼 통신
- 부트스트랩 로더 등에 활용 가능

### 4.5 디버그 신호

| 신호명 | 설명 |
|--------|------|
| /TRST | 테스트 리셋 |
| TCK / DAP0 | 테스트 클록 / Device Access Pin 0 |
| TMS / DAP1 | 테스트 모드 셀렉트 / Device Access Pin 1 |
| TDI / P21.6 | 테스트 데이터 입력 |
| TDO / P21.7 | 테스트 데이터 출력 |
| /TESTMODE / P20.2 | 테스트 모드 셀렉트 입력 |

| D107 | 파란색 | 범용 토글 LED | P13.0 |
| D108 | 파란색 | 범용 토글 LED | P13.1 |
| D109 | 파란색 | 범용 토글 LED | P13.2 |
| D110 | 파란색 | 범용 토글 LED | P13.3 |

### 5.2 버튼/스위치

| 스위치 | 기능 | 위치 |
|--------|------|------|
| S101 | /PORST (리셋) | 하단면 |
| S102 | WAKE (웨이크업) | 하단면 |

- 리셋 버튼(S101): 수동 Power-On Reset 실행
- 웨이크 버튼(S102): TLF35584 웨이크업

### 5.3 CAN (MultiCAN)

- CAN 트랜시버: TLE 7250 GVIO (High Speed)
- MultiCAN Node 0에 연결
- 커넥터: IDC10 (X202)
- TX: P20.8, RX: P20.7

### 5.4 LIN

- LIN 트랜시버: TLE 7259-3GE
- ASCLIN1에 연결
- 커넥터: IDC10 (X203)
- TX: P15.0, RX: P15.1

### 5.5 이더넷

- PHY: PEF7071 (Lantiq Gigabit)
- 커넥터: RJ45 (X204)
- RMII 인터페이스 사용
- **TC237L은 이더넷 미지원 -- 관련 부품 미실장**

| 신호 | 핀 |
|------|-----|
| MDC | P21.2 |
| MDIO | P21.3 |
| TXD1 | P11.2 |
| TXD0 | P11.3 |
| TXEN | P11.6 |
| RXD1 | P11.9 |
| RXD0 | P11.10 |
| CRSDIV | P11.11 |
| REFCLK | P11.12 |

### 5.6 XGA 디스플레이 (320x240)

- 컨트롤러: ILI9341
- SPI-Parallel 변환기(U201)를 통해 MCU와 연결
- SPI 클록: SCLK0 (P20.11)
- 칩셀렉트: SLSO08 (P20.6)

#### 쓰기 (단일)
- 32비트 SPI 전송 (SLSO08 Low 유지)
- Bit31=0 (쓰기), Bit30=0 (단일), Bit29~22=레지스터 번호, Bit21~6=16비트 값, Bit5~0=더미

#### 쓰기 (연속/고속)
- 첫 10비트: Bit9=0 (쓰기), Bit8=1 (연속), Bit7~0=레지스터 번호
- 이후 16비트 값 연속 전송 (첫 번째는 동작 없음, 마지막 후 더미 전송 필요)
- 연속 모드 종료: SLSO08 High 상태에서 SCLK0 상승 에지

#### 읽기 (단일)
- 첫 16비트: Bit15=1 (읽기), Bit14=0 (단일), Bit13~7=레지스터 번호
- 두 번째 26비트: Bit25~16=더미(레지스터 읽기), Bit15~0=더미(값 전송)

### 5.7 터치 컨트롤러

- IC: ADS7843
- QSPI0 + SLSO9 (P20.3)에 연결
- 인터럽트: REQ9 (P20.0, Active)

### 5.8 Micro SD 카드

- Mini SD 카드 슬롯
- **SPI 모드만 지원**
- QSPI1 연결, 칩셀렉트: SLSO19 (P10.5)
- SCLK1: P10.2, MRST1A: P10.1, MTSR1: P10.3

### 5.9 RTC (실시간 클록)

| 항목 | TC237 | TC267/TC277/TC297 |
|------|-------|-------------------|
| IC | MCP79511 | MCP79411 |
| 인터페이스 | QSPI3 | I2C (P15.4/P15.5) |
| 전원 | TLF35584 대기 전압 | TLF35584 대기 전압 |

- 배터리 백업 (BT101): TLF35584 비전원 시 RTC 유지
- 알람 인터럽트: SCU_REQ8 (P33.7, Active High)
- 알람으로 보드 웨이크업(전원 on) 가능
- **EUI-48 고유 ID 내장** -- 이더넷 MAC 주소로 활용 가능

### 5.10 비퍼

- P33.0에 연결
- 2048Hz 주파수 출력 필요

### 5.11 코어 전류 측정

- 50mOhm 저항을 통한 코어(+1.3V) 전류 측정
- **TC237**: AN5와 AN17의 동기 측정
- **기타 CPU**: AN6와 AN22의 동기 측정
- 계산식: `Icore = (VAN_high - VAN_low) / 0.05`
- **반드시 동기 측정 필요**

## 10. 펌웨어 개발 시 핵심 참고사항

1. **전압 레벨 주의**: TC237은 3.3V, TC267/TC277/TC297은 5V 포트 전압 사용
2. **LED 제어**: P13.0~P13.3은 **Low Active** -- GPIO를 Low로 설정해야 LED 점등
3. **TLF35584 초기화 필수**: QSPI2 (P14.2 CS)를 통해 워치독 및 전원 상태 설정
4. **디스플레이 SPI 프로토콜**: 표준 SPI가 아닌 CPLD 기반 스테이트머신 프로토콜 사용
5. **SD 카드**: SPI 모드만 지원, QSPI1 사용
6. **RTC 인터페이스**: TC237은 QSPI3, 기타는 I2C -- CPU에 따라 드라이버 분기 필요
7. **이더넷 MAC**: RTC IC 내장 EUI-48 ID를 MAC 주소로 활용 가능
8. **코어 전류 측정**: ADC 동기 측정 필수 (TC237: AN5/AN17, 기타: AN6/AN22)
9. **외부 디버거 사용 시**: miniWiggler JDS를 반드시 비활성화 (ACTIV LED 꺼짐 확인)
10. **비퍼**: P33.0에 2048Hz 구형파 출력으로 구동
11. **보드 V1.0 하드웨어 이슈**: R147, R148, R337이 와이어랩으로 실장됨
