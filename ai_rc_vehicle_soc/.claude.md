### 핵심 개념 정리 (임베디드 개발자용)
```
피지컬 AI 구조: 센서(카메라) → AI 모델(판단) → 액추에이터(행동)

AI 모델 = 입력 → 함수 → 출력
학습 = 그 함수의 파라미터를 데이터로 자동 튜닝하는 것
C로 비유: 룩업테이블을 데이터로 자동 생성하는 것

기존 방식: if (temperature > 80) alarm = TRUE;  (사람이 규칙을 짬)
AI 방식:   result = ai_model_predict(data);      (데이터가 규칙을 만듦)
```
